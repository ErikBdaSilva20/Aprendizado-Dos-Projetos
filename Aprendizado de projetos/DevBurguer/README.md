# 🎓 Plano de Estudos do Projeto DevBurguer

Olá! Bem-vindo à sua trilha de aprendizado para o projeto **DevBurguer**. Este guia foi criado para te ajudar a dominar as tecnologias e a arquitetura desta aplicação Full-Stack. Para facilitar, dividimos o conteúdo em módulos separados por tecnologias e áreas de atuação (BackEnd e FrontEnd).

## 📂 Como Estudar este Projeto

O conhecimento está organizado em duas partes principais dentro da pasta `PROJETO`:

1.  **Backend**: Onde vivem a lógica de negócios, os bancos de dados e as integrações externas.
2.  **Frontend**: Onde construímos a interface do usuário (UI) e a experiência de navegação (UX).

Sugerimos que você siga a ordem abaixo para um aprendizado mais fluido:

---

### 🔙 Módulo 1: BackEnd (Coração da Aplicação)

Comece entendendo como os dados são processados e armazenados.
*   [**Node.js & Express**](./backend/express.md): A fundação do servidor e como as rotas funcionam.
*   [**PostgreSQL & Sequelize**](./backend/postgresql-sequelize.md): Como modelamos dados estruturados (usuários e produtos).
*   [**MongoDB & Mongoose**](./backend/mongodb-mongoose.md): Por que usamos um banco não-relacional para os pedidos.
*   [**Firebase Admin**](./backend/firebase-admin.md): Segurança e autenticação no lado do servidor.
*   [**Integrações Externas**](./backend/mercadopago.md): Como funcionam as integrações com Mercado Pago e Cloudinary.

---

### 🎨 Módulo 2: FrontEnd (Experiência do Usuário)

Agora, veja como transformamos dados em uma interface incrível.
*   [**React & Vite**](./frontend/react.md): O framework principal e a ferramenta de build rápida.
*   [**Context API**](./frontend/context-api.md): Como gerenciamos o carrinho e os dados do usuário globalmente.
*   [**Estilização (Styled Components)**](./frontend/styled-components.md): A mágica do CSS-in-JS.
*   [**Roteamento (React Router)**](./frontend/react-router-dom.md): Como as páginas se conectam.
*   [**Formulários & Validação**](./frontend/react-hook-form-yup.md): Capturando dados de forma segura com React Hook Form e Yup.

---

## 🚀 Como Explorar os Arquivos

Cada arquivo `.md` nesta pasta segue uma estrutura pedagógica:
- **Conceitos Principais**: Termos técnicos essenciais.
- **Explicação Simplificada**: O que é aquilo, na "vida real".
- **Por que isso existe**: O problema que essa tecnologia resolve.
- **Exemplos de Uso no Projeto**: Onde encontrar no código real do DevBurguer.
- **Dicas & Desafios**: Para você testar seu conhecimento.

Aproveite bem a sua semana de estudos! Se tiver dúvidas, consulte a documentação oficial linkada em cada arquivo ou me pergunte! 🍔🚀
