# 🔙 Backend (Node.js & API)

Nesta pasta, você encontrará os detalhes de como o coração da nossa lanchonete (o servidor) foi construído. O backend do DevBurguer é responsável por:
- Processar as regras de negócio.
- Gerenciar os dois bancos de dados (PostgreSQL e MongoDB).
- Validar a segurança e autenticação do Firebase.
- Gerar pagamentos no Mercado Pago.
- Fazer upload de imagens pro Cloudinary.
