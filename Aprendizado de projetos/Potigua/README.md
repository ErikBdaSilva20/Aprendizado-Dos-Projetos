# 🎓 Guia de Aprendizado: Projeto Potigua (Presentes de Casamento)

Bem-vindo ao seu guia de estudos personalizado para o projeto **Potigua**! 🚀

Este projeto é uma aplicação **Full-Stack** (Frontend + Backend) para gerenciamento de uma lista de presentes de casamento. Ele utiliza tecnologias modernas e robustas para garantir performance e facilidade de manutenção.

Este diretório foi criado para ser o seu **professor particular**. Aqui, cada tecnologia é desmembrada para que você entenda não apenas *como* funciona, mas *por que* foi escolhida.

## 🛠️ Tecnologias Utilizadas

### Frontend
- **React 19**: Biblioteca para construção de interfaces.
- **Vite**: Ferramenta de build ultra-rápida.
- **Styled-Components**: CSS-in-JS para componentes elegantes.
- **React-Router-Dom**: Navegação entre páginas (Home, Login, Admin).
- **Axios**: Cliente HTTP para falar com o servidor.

### Backend
- **Node.js**: Ambiente de execução JavaScript no servidor.
- **Express**: Framework para criação de APIs.
- **MongoDB + Mongoose**: Banco de dados NoSQL e modelagem de dados.
- **Cloudinary**: Serviço de armazenamento de imagens.
- **Multer**: Middleware para lidar com uploads de arquivos.
