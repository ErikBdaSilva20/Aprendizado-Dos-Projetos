# 🧠 Central de Aprendizado - My Knowledge Hub

Bem-vindo(a) ao meu repositório pessoal de estudos! Este espaço foi criado para centralizar, organizar e documentar tudo o que venho aprendendo no mundo do desenvolvimento de software.

## 📌 O que é este projeto?

Este repositório serve como um "Cérebro Digital" (Zettelkasten adaptado para código). Aqui, utilizo a metodologia **LearnAgent** para garantir que o conhecimento não seja apenas consumido, mas processado e documentado de forma clara para futuras consultas.

> "Se algo precisou ser pesquisado ou gerou dúvida → **deve ser documentado**."

## 📖 Como a documentação é organizada?

Seguindo o padrão do **LearnAgent**, cada tópico de estudo dentro dos arquivos `.md` segue esta estrutura:

1.  **Conceitos Principais**: O que é a tecnologia.
2.  **Explicação Simplificada**: Resumo direto ao ponto.
3.  **Por que isso existe?**: Qual problema real isso resolve.
4.  **Exemplos de Uso**: Trechos de código comentados.
5.  **Dificuldades & Soluções**: Notas sobre o que foi difícil e como resolvi.
